🚀 Créer une affichette rapidement
1
Scanner ou saisir le code produit
Utilisez le bouton 📷 pour scanner le code-barres ou tapez manuellement. Le produit est chargé automatiquement si la BASE est renseignée.
2
Vérifier les informations
Désignation FR/AR, prix, promotion, grammage. Corrigez si nécessaire. La prévisualisation A7 se met à jour en temps réel.
3
Choisir le template
Sélectionnez le gabarit adapté (prix choc, promo, nouveauté…). L'aperçu reflète immédiatement votre choix.
4
Ajouter à la file d'impression
Cliquez Ajouter à la file. Accumulez plusieurs affichettes, puis imprimez-les toutes en PDF depuis l'onglet FILE.
⌨️ Raccourcis clavier
Entrée Valider le code F5 Rafraîchir l'aperçu
📦 Gérer la base produits & templates
▶
Onglet BASE
Consultez, ajoutez, modifiez et supprimez des produits. Importez depuis un fichier CSV ou Excel. Exportez la base complète.
▶
Onglet TEMPLATES
Créez et personnalisez vos gabarits d'affichettes : couleurs, polices, zones, logos rayon. Chaque template peut être dupliqué et renommé.
▶
Onglet HISTORIQUE
Retrouvez toutes les affichettes générées, filtrez par date, opérateur, code produit. Régénérez rapidement une affichette passée.
▶
Onglet CONFIG → Mapping
Associez les colonnes de votre fichier d'import aux champs de l'application (code, prix, désignation…).
💡 Conseil : Préparez un fichier CSV avec les colonnes : code, designation_fr, designation_ar, prix, promo. L'import batch est le moyen le plus rapide de peupler la base.
🔐 Administration & sécurité
▶
Gestion des utilisateurs
Bouton Admin dans le header → créez des comptes, définissez les rôles et permissions par onglet. Le mot de passe est hashé (FNV-1a).
▶
Compte administrateur par défaut
Login : admin / Mot de passe : marjane2026.
▶
Sauvegarde des données
Toutes les données (base, historique, templates) sont désormais stockées dans IndexedDB (capacité illimitée, dans le navigateur). La file et la config légère restent dans localStorage. Exportez régulièrement via CONFIG → Export.
▶
Réinitialisation
En cas de problème d'authentification, la page de connexion propose un lien de réinitialisation. Cette action efface tous les comptes.
⚠️ Important : Les données sont locales au navigateur de chaque poste. Pour partager la base entre plusieurs postes, utilisez l'export/import JSON ou configurez un serveur partagé.
🖨️ Générer des affichettes au format A4
1
Accéder à l'onglet A4
Cliquez sur l'onglet A4 dans la barre de navigation. L'interface de composition grand format s'affiche avec une prévisualisation en temps réel.
2
Saisir les informations produit
Renseignez le code produit, la désignation FR/AR, le prix normal et promotionnel, le grammage et la mention fidélité si applicable.
3
Choisir le template A4
Sélectionnez un gabarit A4 adapté : fond coloré, affichette double, plein-page prix ou promo spéciale. L'aperçu se met à jour instantanément.
4
Exporter en PDF A4
Cliquez Télécharger PDF pour générer un fichier prêt à imprimer en format A4 (210×297 mm). Compatible toutes imprimantes.
5
Ajouter à la file d'impression
Ajoutez plusieurs affichettes A4 dans la file, puis imprimez le lot complet en un seul PDF multi-pages depuis l'onglet FILE.
⚡ Différences A7 vs A4
📌 FORMAT A7
• 74×105 mm
• Étiquette rayon
• 4 affichettes/page
• Template POTX
🆕 FORMAT A4
• 210×297 mm
• Affiche vitrine / tête de gondole
• 1 affichette/page
• Export PDF direct
💡 Conseil : Utilisez le format A4 pour les têtes de gondole, les promotions flash et les entrées de rayon. Pour les étiquettes de gondole classiques, privilégiez le format A7.
📱 Utiliser comme application installée
📲
Installation sur Android (Chrome)
Cliquez le bouton Installer dans le header, ou menu Chrome ⋮ → Ajouter à l'écran d'accueil. L'app s'ouvre en plein écran sans barre de navigateur.
🍎
Installation sur iOS (Safari)
Appuyez sur le bouton Partager ↑ puis Sur l'écran d'accueil. L'app est ajoutée comme une vraie application.
🖥️
Installation sur PC (Chrome/Edge)
Cliquez le bouton Installer ou l'icône ⊕ dans la barre d'adresse. L'app s'ouvre dans sa propre fenêtre.
📶
Mode hors ligne
Une fois installée, l'application fonctionne sans connexion internet. La base, les templates et l'historique restent accessibles.
🔄
Mises à jour
Une bannière apparaît en haut de l'écran lorsqu'une nouvelle version est disponible. Cliquez Mettre à jour pour l'appliquer.
